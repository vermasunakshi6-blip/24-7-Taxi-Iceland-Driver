# Leggoo Driver

**Standalone Driver app.** No dependency on the main Leggoo project. All code and assets are inside this project.

- **App name:** Leggoo Driver  
- **Application ID:** com.ride.app.driver  

## Setup

```bash
flutter pub get
```

## Run

```bash
flutter run
```

## Build release APK

```bash
flutter build apk
```

Can be installed alongside Leggoo Rider. Use your own Firebase/Google config in `android/` and `lib/config/` as needed.
